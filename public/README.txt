Restaurant Menu App - Public Assets
====================================

Icons and Favicons
------------------
All icons are configured and ready to use!

Files in this folder:
- favicon.ico - Browser favicon (16x16, 32x32)
- apple-touch-icon.png - iOS home screen icon (180x180)
- icon-192.png - PWA icon (192x192)
- icon-512.png - PWA icon (512x512)
- icon-192-maskable.png - PWA maskable icon (192x192)
- icon-512-maskable.png - PWA maskable icon (512x512)
- manifest.json - PWA manifest configuration

Configuration:
--------------
Icons are automatically referenced in:
✓ app/layout.tsx - Next.js metadata
✓ manifest.json - PWA configuration

The app will use these icons for:
- Browser tabs (favicon.ico)
- iOS home screen (apple-touch-icon.png)
- Android/PWA app icons (icon-192.png, icon-512.png)
- Adaptive icons (icon-*-maskable.png)

Customization:
--------------
To use your own restaurant logo:
1. Replace the PNG files with your logo (maintain same sizes)
2. Ensure favicon.ico includes 16x16 and 32x32 sizes
3. Optionally update theme color in app/layout.tsx

The icons are working correctly as-is!
